# 🍅 ReDiary - 일기를 작성며 자라나는 토마토 다이어리
### 기능
1. 일기를 작성하면 토마토가 자라난다
2. 일기 작성할 때마다 스티커를 획득할 수 있다
3. 모은 스티커들을 다시 확인할 수 있다.
4. 작성한 일기들은 로컬 스토리지에 저장되어 창을 닫은 후에도 기록이 남아있다.

### 기술스택
- HTML / CSS / JS
- Github pages 배포
